### 协议

 协议 | 体现 | 名字
---|---|---
应用层| FTP|  
传输层| TCP | 段
网络层| IP | 数据报
链路层| 以太网 | 帧

* 帧 --> IP / ARP / RARP  
* IP --> TCP / UDP / ICMP / IGMP
* TCP/UDP --> 应用程序
* 
* IP/ARP/RARP 封装成帧, IP属于网络层, 其他属于链路层
* TCP/UDP/ICMP/IGMP 封装成数据报, TCP/UDP属于传输层, 其他属于网络层
